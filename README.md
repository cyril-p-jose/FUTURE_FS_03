## ✨ Overview

**Harbor Brew Café** is a modern, responsive, and visually rich landing page built as part of a Full Stack Web Development internship project.  

It showcases a premium UI/UX experience inspired by Kerala’s riverside beauty — combining nature, coffee culture, and interactive web design.

---

## 🚀 Key Highlights

- 🌊 Kerala-inspired riverside café UI
- 📱 Fully responsive (mobile-first design)
- 🍔 Interactive hamburger navigation
- 🎯 Smooth scrolling navigation system
- 🖼️ Image gallery with lightbox feature
- ✨ Scroll-based reveal animations
- 📍 Embedded Google Maps integration
- 💬 Customer testimonials section
- 📞 WhatsApp quick contact button
- 🎨 Modern gradient + glassmorphism styling
